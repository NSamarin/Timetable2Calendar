How to use

  1. Login to www.myed.ed.ac.uk
  2. Go to Studies -> Path -> Launch Path
  3. Add all courses into the system
  4. Choose to view Timetable for Semester 1 (only)
  5. Copy the link and paste it into the form above

Notes

  1. You can generate the timetable only for one semester.
  2. Generated timetable will ONLY include lectures.
  3. Although this algorith was tested on numerous timetables, it is still advised to double check and to compare generated timetable with your original timetable. Please contact us if you find any errors/bugs.
